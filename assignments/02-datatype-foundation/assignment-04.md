# Data Type Mix-Bag (Real World Integration)

**Goal:** In sabhi types ko ek saath use karke ek complete feature ka logic banayein.

## 1. User Profile Database: Student Identity Card

**Required Knowledge:** Strings, Numbers, Booleans.

Ek student ka data store karein:

- name: "Amit Kumar" (String)
- age: 20 (Number)
- isGraduated: false (Boolean)
- graduationYear: undefined (Kyunki abhi graduation nahi hui)

```js
// name
// age
// isGraduated
// graduationYear
console.log() // all things in one string
```

> Real-life Use Case: Student Management System.
